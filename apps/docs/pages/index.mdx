## 主要功能

### 网络诊断工具

idcd 提供全球多节点拨测服务，帮助你快速定位网络问题：

- **HTTP 拨测** - 检测网站可达性、响应时间、状态码
- **Ping 测试** - 测试延迟、丢包率
- **TCPing** - TCP 端口连通性测试
- **DNS 查询** - 多地 DNS 解析检测
- **路由追踪** - 查看数据包路径

### Evidence 证据服务

为你的网络诊断结果提供可信证据，支持：

- 实时监控告警
- 历史数据查询
- 报告生成与分享
- API 接口集成

## 开发者工具

提供实用的在线开发工具：

- JSON 格式化
- Base64 编解码
- 时间戳转换
- Hash 计算
- JWT 解码
- 正则表达式测试
- Cron 表达式解析
- 二维码生成
- CIDR 计算器
- IPv6 转换
