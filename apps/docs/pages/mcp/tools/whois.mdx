# whois Tool

查询域名或 IP 地址的 WHOIS 注册信息，包括注册商、注册日期和到期时间。

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `query` | string | ✓ | — | 要查询的域名或 IP 地址 |

## 示例

**域名查询**：

```json
{"query": "github.com"}
```

**输出**：

```
WHOIS: github.com
注册商: MarkMonitor Inc.
注册日期: 2007-01-26
到期日期: 2026-01-26
```

**IP 地址查询**：

```json
{"query": "8.8.8.8"}
```

**输出**：

```
WHOIS: 8.8.8.8
注册商: ARIN
注册日期: 1992-12-01
```

## 无 API Key 时的输出

```
⚠ 需要 API key，请设置 IDCD_API_KEY 环境变量
```

## 返回字段说明

| 字段 | 说明 |
|------|------|
| 注册商 | 域名注册机构名称 |
| 注册日期 | 域名初次注册时间 |
| 到期日期 | 域名当前注册有效期截止日期 |

部分字段如 WHOIS 数据中缺失则不显示。

## 常见使用场景

- 确认域名注册人和注册商
- 检查域名是否即将到期
- 查询 IP 段所属机构
- 验证竞品域名注册信息

## API 端点

内部调用 `GET /v1/info/whois?q={query}`，认证方式为 `Authorization: Bearer <key>`。
