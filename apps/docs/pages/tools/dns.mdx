# DNS 查询

DNS 查询工具用于检测域名在全球各地的 DNS 解析结果，帮助发现 DNS 污染、解析不一致等问题。

## 功能特点

- 多地 DNS 解析对比
- 支持多种记录类型（A、AAAA、CNAME、MX、TXT、NS 等）
- DNS 服务器响应时间测试
- DNSSEC 验证
- 反向 DNS 查询

## 使用方法

### 基础查询

1. 输入域名（例如：`example.com`）
2. 选择记录类型（默认 A 记录）
3. 选择测试节点
4. 点击"开始查询"

### 记录类型说明

- **A 记录**：IPv4 地址
- **AAAA 记录**：IPv6 地址
- **CNAME 记录**：别名记录
- **MX 记录**：邮件服务器记录
- **TXT 记录**：文本记录，常用于域名验证
- **NS 记录**：名称服务器记录
- **SOA 记录**：起始授权机构记录
- **PTR 记录**：反向 DNS 记录

## 结果解读

测试结果包括：

- **解析结果**：域名对应的 IP 地址或记录值
- **TTL**：记录的生存时间（Time To Live）
- **响应时间**：DNS 查询的响应时间
- **DNS 服务器**：使用的 DNS 服务器地址
- **一致性检查**：各节点解析结果是否一致

## 常见问题

### DNS 污染检测

如果不同地区的 DNS 解析结果不一致，可能存在 DNS 污染问题。

### 解析慢怎么办？

- 检查 DNS 服务器响应时间
- 考虑更换更快的 DNS 服务器
- 检查 TTL 设置是否合理

### DNSSEC 是什么？

DNSSEC 是 DNS 安全扩展，用于验证 DNS 解析结果的真实性，防止 DNS 欺骗攻击。
