# WHOIS 查询

WHOIS 查询工具用于查询域名的注册信息，包括注册人、注册时间、到期时间、DNS 服务器等。

## 功能特点

- 域名注册信息查询
- IP 地址归属查询
- 域名到期提醒
- 历史 WHOIS 记录（需登录）

## 使用方法

1. 输入域名或 IP 地址
2. 点击"查询"
3. 查看详细信息

## 结果解读

查询结果包括：

### 域名信息

- **域名**：查询的域名
- **注册商**：域名注册商
- **注册人**：域名所有者（可能被隐私保护屏蔽）
- **注册日期**：域名首次注册时间
- **到期日期**：域名到期时间
- **更新日期**：域名信息最后更新时间
- **DNS 服务器**：域名使用的 DNS 服务器

### IP 信息

- **IP 地址**：查询的 IP 地址
- **所属组织**：IP 地址分配的组织
- **国家/地区**：IP 地址所在地理位置
- **AS 号**：自治系统号
- **网络范围**：IP 地址所在的网络段

## 隐私保护

许多域名注册商提供 WHOIS 隐私保护服务，隐藏注册人的真实信息，显示为代理信息。

## 常见问题

### WHOIS 信息不准确？

WHOIS 信息由注册商维护，如果信息不准确，需要联系域名注册商更新。

### 如何保护隐私？

可以使用域名注册商提供的隐私保护服务，隐藏个人联系信息。

### 域名即将到期怎么办？

及时续费，避免域名被释放。建议开启自动续费功能。
