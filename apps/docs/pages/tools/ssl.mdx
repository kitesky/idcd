# SSL 证书检测

SSL 证书检测工具用于检查网站的 SSL/TLS 证书状态、有效期、加密强度等信息。

## 功能特点

- 证书有效期检测
- 证书链验证
- 加密算法检测
- 证书颁发者信息
- 到期提醒

## 使用方法

1. 输入域名（例如：`example.com`）或完整 URL
2. 点击"开始检测"
3. 查看证书详情

## 结果解读

检测结果包括：

- **证书状态**：有效、即将过期、已过期
- **颁发给**：证书持有者信息
- **颁发者**：证书颁发机构（CA）
- **有效期**：证书的生效日期和到期日期
- **加密算法**：使用的加密算法和密钥长度
- **证书链**：完整的证书信任链
- **支持的协议**：TLS 1.2、TLS 1.3 等

## 常见问题

### 证书即将过期怎么办？

证书到期前应及时续签，建议在到期前 30 天进行更新。

### 证书不受信任的原因

- 使用自签名证书
- 证书链不完整
- 证书已被吊销
- 使用的 CA 不在信任列表中

### TLS 版本选择

- **TLS 1.3**：最新最安全，推荐使用
- **TLS 1.2**：安全，广泛支持
- **TLS 1.1 及更早**：不推荐，存在安全风险
