# 快速开始

## 什么是 idcd？

idcd 是一个多节点网络诊断工具平台，提供全球分布式拨测服务。通过 idcd，你可以：

- 从全球多个节点测试你的服务可达性
- 获取真实的网络延迟和性能数据
- 快速定位网络问题的地理位置
- 生成可信的网络诊断证据

## 主要特点

### 全球节点覆盖

idcd 在全球多个地区部署了测试节点，包括：

- 亚洲：中国大陆、香港、日本、新加坡
- 北美：美国东西海岸
- 欧洲：英国、德国、法国
- 其他：澳大利亚、南美等

### 多种测试方式

- **HTTP/HTTPS 拨测** - 检测网站可用性、响应时间、SSL 证书状态
- **Ping 测试** - ICMP 延迟和丢包率测试
- **TCPing** - TCP 端口连通性和响应时间
- **DNS 查询** - 多地 DNS 解析结果对比
- **Traceroute** - 网络路径追踪，定位延迟节点

### 一键诊断

通过一键诊断功能，同时执行多种测试，快速获得全面的网络健康报告。

## 使用流程

### 1. 选择测试工具

访问 [idcd.com](https://idcd.com)，从首页选择你需要的诊断工具。

### 2. 配置测试参数

输入测试目标（URL、IP 地址、域名等）和相关参数。

### 3. 选择测试节点

选择你想要测试的地理节点，或者选择"全部节点"进行全球测试。

### 4. 查看结果

测试完成后，查看详细的测试结果，包括：

- 响应时间
- 成功率
- 错误信息
- 节点对比

### 5. 保存与分享

可以保存测试结果，生成分享链接，或导出报告。

## 注册账户（可选）

虽然大部分功能可以匿名使用，但注册账户后你可以：

- 保存测试历史
- 设置定期监控
- 使用 API 集成
- 获取 Evidence 证据服务
