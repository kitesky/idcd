# ip Tool

查询 IPv4 或 IPv6 地址的地理归属、ASN 和 ISP 信息。

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `address` | string | ✓ | — | 要查询的 IPv4 或 IPv6 地址 |

## 示例

**IPv4 请求**：

```json
{"address": "8.8.8.8"}
```

**输出**：

```
IP: 8.8.8.8
ASN: AS15169
位置: Mountain View, United States
ISP: Google LLC
```

**IPv6 请求**：

```json
{"address": "2001:4860:4860::8888"}
```

**输出**：

```
IP: 2001:4860:4860::8888
ASN: AS15169
位置: United States
ISP: Google LLC
```

**中国 IP**：

```json
{"address": "114.114.114.114"}
```

**输出**：

```
IP: 114.114.114.114
ASN: AS4808
位置: 南京, China
ISP: China Unicom
```

## 无 API Key 时的输出

```
⚠ 需要 API key，请设置 IDCD_API_KEY 环境变量
```

## 返回字段说明

| 字段 | 说明 |
|------|------|
| IP | 查询的 IP 地址（原样返回） |
| ASN | 自治系统号（如 AS15169）|
| 位置 | 城市和国家（`城市, 国家` 格式） |
| ISP | 互联网服务提供商或组织名称 |

## 常见使用场景

- 验证 CDN 节点 IP 归属
- 排查访客来源地区
- 检查自建 IP 是否被标记为异常 ASN
- 确认出口 IP 的运营商归属

## API 端点

内部调用 `GET /v1/info/ip?q={address}`，认证方式为 `Authorization: Bearer <key>`。
