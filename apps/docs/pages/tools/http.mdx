# HTTP 拨测

HTTP 拨测工具用于检测网站在全球各地的可达性、响应时间和状态码。

## 功能特点

- 支持 HTTP 和 HTTPS 协议
- 自定义请求方法（GET、POST、HEAD 等）
- 自定义请求头和请求体
- SSL 证书检测
- 响应时间统计
- 状态码检测

## 使用方法

### 基础测试

1. 输入目标 URL（例如：`https://example.com`）
2. 选择测试节点
3. 点击"开始测试"

### 高级选项

- **请求方法**：选择 HTTP 方法（GET、POST、PUT、DELETE 等）
- **请求头**：添加自定义请求头，例如 `User-Agent`、`Authorization` 等
- **请求体**：对于 POST/PUT 请求，可以添加请求体数据
- **超时设置**：设置请求超时时间（默认 10 秒）
- **跟随重定向**：是否跟随 HTTP 重定向

## 结果解读

测试结果包括以下信息：

- **响应时间**：从发起请求到接收完整响应的时间
- **状态码**：HTTP 响应状态码（200、404、500 等）
- **响应头**：服务器返回的响应头信息
- **证书信息**：对于 HTTPS，显示 SSL 证书详情
- **错误信息**：如果测试失败，显示错误原因

## 常见问题

### 为什么某些节点测试失败？

可能的原因：

- 目标服务器在该地区不可达
- 防火墙或 CDN 策略限制
- 目标服务器过载
- 网络故障

### 如何解读响应时间？

- **< 100ms**：优秀
- **100-300ms**：良好
- **300-1000ms**：一般
- **> 1000ms**：较慢，需优化
